\# STM32 Linked LED



\## Description

Simple linked LED project using STM32 HAL.



\## Features

\- LED blinking

\- GPIO output/input control handling

\- push button LED control

&#x20; 

\## Hardware

\## Toolchain

\- STM32CubeIDE

\- HAL Driver
